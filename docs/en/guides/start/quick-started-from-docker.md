# Quick Start (Docker)

## 0. Prerequisites
Make sure you have:

- Installed Docker

## 1. Run the Project

```bash
docker run -p 3000:3000  tachybase/tachybase-all:latest
```

## 2. Access Tachybase

Open [http://localhost:3000](http://localhost:3000) in your browser. The initial account and password are `admin@tachybase.com` and `!Admin123.`.
