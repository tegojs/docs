# Markdown Block

## Introduction

Markdown blocks do not need to be bound to a data source. Using Markdown syntax to define text content, they can be used to display formatted text content.

## Add Block

Markdown blocks can be added to pages or modals
![](/other-blocks/markdown-add.png)

Inline Markdown blocks can also be added to form blocks and details blocks
