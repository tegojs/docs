## Introduction

The save data operation allows specifying the value to be saved for fields through field assignment, with higher priority than values filled in the form. It can also be combined with workflows to implement automated data processes.

## Operation Configuration Items

### Field Assignment

If field assignment is configured, the same field will use the value configured in field assignment, with higher priority than the value filled in the form.

![20240423213245](/actions/save-record-3.png)
