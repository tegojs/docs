# Filter

## Introduction

Filter operations usually appear in data blocks. By configuring various conditions to filter data, it should be noted that if the block has already configured a data scope, the filter operation will be merged with the data scope conditions as the final filter conditions.

### Filterable Fields

By default, all fields of the current table are enabled. Only enabled fields can be selected in the condition field list.

Relationship table fields can also be used as filter fields.
