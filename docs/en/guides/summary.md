# Basic Features

> The following features are currently supported

- Undo/Redo
- Add rows/columns
- Delete rows/columns
- Hide rows/columns
- Dynamic row/column resizing
- Freeze rows/columns
- Add borders/colors/border line styles
- Merge cells
- Copy
- Paste
- Print
- Formulas
- Filter
- Auto-fill
- Multiple sheet support
- Font styles
- Custom font/size/color
- Fill foreground/background color
- Format validation
- Text alignment
- Text wrapping
