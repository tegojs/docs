# Custom Element Class

This class is a custom dom element class

Calling method: Since it's a custom base class, it's generally the element you create

## Instance Properties

### el

Mounted is the native HTMLElement object generated by the current element

### data

Mount single data object to html, doesn't seem to be used currently, can be customized

## Public Methods

### data(key, value) Set data

Set the value of the data object

### on(eventNames, handler) Event listener

`@param eventNames` Event name

`@param handler` Event

### offset(value) Set offset

Only four properties are useful: left, top, right, bottom

`@param value` Value, example: `{ left: 35 }` No unit needed, value is object {key, value}

### scroll(v) Set scroll

Only two properties are useful: left, top

`@param v` Is object {key, value}

### box() Get element information relative to viewport

Method returns element's size and its position relative to viewport

### parent() Get parent node

Method gets parent node element

### children(...eles) Add multiple child nodes

Can add any number of child nodes

`@param eles` Must be native HTMLElement object, can add multiple

### removeChild(el) Remove child node

`@param el` Must be native HTMLElement object

### child(arg) Add single child node

`@param arg` Can be native HTMLElement object or just string

### contains(ele)

Used to detect whether child nodes in the node returned by the current DOM object contain the specified node, return value is boolean type

`@param ele` Must be native HTMLElement object

### className(v) Get/set class name

Used to set className, when v has a value, if no value returns current className

`@param v` class

### addClass(name) Add class name

`@param name` string

### hasClass(name) Check class name, returns boolean

`@param name` string

### toggle(cls = 'active') Toggle class name

`@param cls` string

### toggleClass(name) Toggle class name

`@param name` string

### active(flag = true, cls = 'active') Set active class name

`@param flag` Boolean

`@param cls` string

### checked(flag = true) Set checked/active

`@param flag` Boolean

### disabled(flag = true) Set non-editable

`@param flag` Boolean

### attr(key, value) Set attribute

`@param key` string

`@param value` string/number

### removeAttr(key) Remove attribute

`@param key` string

### html(content) Set/get html value

With parameter sets html, without parameter gets html value

`@param content` string

### val(v) Set value

With parameter sets value, without parameter gets value

### focus() Get focus

Automatically get focus

### cssRemoveKeys(...keys) Remove css

Remove css through key method

`@param keys` string

### css(name, value) Set style css

`@param name` string

`@param value` string/number

### computedStyle() Get computed style

### show() Show current element

### hide() Hide current element
