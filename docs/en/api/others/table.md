# Table Class

This class is mainly responsible for table drawing and rendering

::: tip New Version
A separate data object is created for each individual worksheet, so you should now find the corresponding data object in instance.datas and then operate
:::

Calling method:

```js
// instance is the instance you created, public methods can be called directly, private methods cannot
instance.sheet.table.publicFn(args);
```

## Instantiation

Need to pass proxy data_proxy generated object, need custom Element, not

```js
const table = new Table(el, data);
```

## Instance Properties

### el

Custom Element mounted to current table, not HTMLElement

### draw

Mount drawing tool class, build bottom layer for canvas

### data

Mount proxy data_proxy generated object

**Note**: Not binding value

## Private Methods

### renderAutofilter(viewRange)

**Function**: Render auto filter function

`@param viewRange` Viewport range, Object type

### renderCell(draw, data, rindex, cindex, yoffset = 0)

**Function**: Render cell

`@param draw` Draw canvas tool class

`@param data` Object generated by data_proxy

`@param rindex` Row coordinate, starts from 0

`@param cindex` Column coordinate, starts from 0

`@param yoffset` Y-axis offset

### renderContent(viewRange, fw, fh, tx, ty)

**Function**: Render text

`@param viewRange` Viewport range, Object type

`@param fw` Left index bar width

`@param fh` Table top title bar height

`@param tx` If frozen rows and columns exist in table, tx is total width of frozen rows

`@param ty` If frozen rows and columns exist in table, ty is total height of frozen rows

### renderContentGrid({ sri, sci, eri, eci, w, h }, fw, fh, tx, ty)

**Function**: Render grid lines

`@param sri` Start row coordinate

`@param sci` Start column coordinate

`@param eri` End row coordinate

`@param eci` End column coordinate

`@param w` Width

`@param h` Height

`@param fw` Left index bar width

`@param fh` Table top title bar height

`@param tx` If frozen rows and columns exist in table, tx is total width of frozen rows

`@param ty` If frozen rows and columns exist in table, ty is total height of frozen rows

### renderFixedHeaders(type, viewRange, w, h, tx, ty)

**Function**: Render title bar and left index bar style

`@param type` type: all | left | top

`@param w` Width

`@param h` Height

`@param tx` If frozen rows and columns exist in table, tx is total width of frozen rows

`@param ty` If frozen rows and columns exist in table, ty is total height of frozen rows

### renderFixedLeftTopCell(fw, fh)

**Function**: Render cell at intersection of title bar and index bar upper left corner

`@param fw` Left index bar width

`@param fh` Table top title bar height

### renderFreezeHighlightLine(fw, fh, ftw, fth)

**Function**: Render freeze row and column highlight line

`@param fw` Left index bar width

`@param fh` Table top title bar height

`@param ftw` If frozen rows and columns exist in table, ftw is total width of frozen rows

`@param fth` If frozen rows and columns exist in table, fth is total height of frozen rows

### renderSelectedHeaderCell(x, y, w, h)

**Function**: Render selected title bar and index bar style

`@param x` Calculated value distance from left

`@param y` Calculated value distance from top

`@param w` Column width

`@param h` Row height

## Public Methods

### resetData(data)

**Function**: Refresh data

`@param data` data_proxy object

### render()

**Function**: Render table

Usage:

```js
const table = new Table(el, data);
table.render();
```

### clear()

**Function**: Clear table drawing canvas area

```js
const table = new Table(el, data);
table.clear();
```

## Others

### tableFixedHeaderStyle()

**Function**: Get default configured table title bar and index bar selected style

### tableGridStyle

Default grid style declaration

### cellPaddingWidth

Cell default padding

### tableFixedHeaderCleanStyle

Default fill style
